# COBO LitVM Testnet Lab v1.4

Educational Web3 testnet lab on LitVM LiteForge.

## COBO Research Credit (testnet only)

COBO is used inside this lab as a symbolic Research Credit and Proof-of-Learning marker. It has no monetary value, cannot be redeemed, is not for sale, is not an airdrop, and is not an official LitVM token.
