# COBO LitVM Testnet Lab v1.3

Laboratório educacional para aprender Web3 na LitVM LiteForge Testnet.

## Objetivo

Documentar interações práticas com MetaMask, zkLTC de teste, contratos, DeFi simples, swaps e explorer.

Este projeto não vende token, não promete lucro e não usa dinheiro real.

## Rede

- Nome: LitVM LiteForge Testnet
- Chain ID: 4441
- Token nativo: zkLTC
- RPC: https://liteforge.rpc.caldera.xyz/http
- Explorer: https://liteforge.explorer.caldera.xyz
